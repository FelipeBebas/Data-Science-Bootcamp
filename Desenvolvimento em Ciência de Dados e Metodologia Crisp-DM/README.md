## 1) Módulo: Desenvolvimento em Ciência de Dados e Metodologia Crisp-DM
Este módulo contém entregas relacionadas ao tema 'Desenvolvimento em Ciência de Dados e Metodologia Crisp-DM'.
O CRISP-DM (Cross-Industry Standard Process for Data Mining) é um método usado para mineração e ciência de dados divido em 6 etapas: Entedimento do negócio - Entendimento de dados - Preparação de dados - Modelagem - Avaliação - Implantação
